# Recipe generator App 🍔

 [Click here to view the  Live demo](https://saladilakshman.github.io/recipe-app/).

## Project Description
A recipe generator webapp created using edamam API where the user can search for recipes and can download the recipe steps.Recipes includes cookies, salads, biryani,pasta,drinks,sweets,fish and so on.

### `Concepts used`
Blob: A method used to deal with large objects such as images,audio,video and text files and can be used to share and download them.

Fetch API: A built-in javascript method used to fetch the api data.
